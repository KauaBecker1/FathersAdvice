## Qualidade de Código

Como pedido em sala de aule, O projeto utiliza PMD e Checkstyle para garantir a qualidade do código.

### Executar ambas as verificações *(PMD e Checkstyle)*

```bash
mvn validate
```

### Gerar relatórios

```bash
mvn site
```

Os relatórios serão gerados na pasta `target/site/`.
